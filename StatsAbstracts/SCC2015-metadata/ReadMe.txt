This package contains metadata for the authors and papers studied in Ji and Jin's paper titled "Coauthorship and Citation Networks for Statisticians" to appear in the Annals of Applied Statistics. Please use these data together with the package available at http://faculty.franklin.uga.edu/psji/sites/faculty.franklin.uga.edu.psji/files/SCC2015.zip
 

The authorMeta.txt file contains author names, citation counts and citer counts; and the paperMeta.txt file contains  paper DOIs,  publication year, title, citation counts and abstracts. 

Please feel to let us know if there are any questions regarding the data  set, the code and the paper. Our email addresses are psji@uga.edu and jiashun@stat.cmu.edu.  Thank you very much. 

Pengsheng Ji and Jiashun Jin
